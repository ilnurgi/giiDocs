psql
====

connect
-------

Подключиться к базу

.. code-block:: sh

	postgres=# \connect ilnurgi_database 


list
----

Список существующих баз

.. code-block:: sh

	postgres=# \list

 	postgres  | postgres 
 	template0 | postgres 
 	template1 | postgres
