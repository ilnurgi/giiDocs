Camera
======

.. py:method:: cameraCapturePicture(str targetPath[ , bool useAutoFocus])

    :param str targetPath: путь к файлу для сохранения
    :param bool useAutoFocus: не обязательный, по умолчанию true, включить автофокус

    получает картинку с фотокамеры и сохраняет её на смарте


.. py:method:: cameraInteractiveCapturePicture(str targetPath)

    включает камеру для получения фотографии, и сохраняет фото по указанному пути
