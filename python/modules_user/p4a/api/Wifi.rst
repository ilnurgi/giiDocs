Wifi
====


.. py:method:: checkWifiState()

    проверяет статус вафли


.. py:method:: toggleWifiState([bool enabled])

    включает или выключает вафлю


.. py:method:: wifiDisconnect()

    отключается от текущей точки доступа


.. py:method:: wifiGetConnectionInfo()

    возвращает информацию о подключении


.. py:method:: wifiGetScanResults()

    возварщает список доступных точек доступа


.. py:method:: wifiLockAcquireFull()

    Acquires a full Wifi lock.


.. py:method:: wifiLockAcquireScanOnly()

    Acquires a scan only Wifi lock.


.. py:method:: wifiLockRelease()

    Releases a previously acquired Wifi lock.


.. py:method:: wifiReassociate()

    Reassociates with the currently active access point.


.. py:method:: wifiReconnect()

    Reconnects to the currently active access point.


.. py:method:: wifiStartScan()

    Starts a scan for Wifi access points.
