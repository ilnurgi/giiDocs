pymongo - драйвер для MongoDB
=============================

MongoDB - высокопроизводительная нереляционная система управления базами данных.

Данный конспект представляет собой вольный перевод и доработка документации официальной документации pymongo_.

Версия pymongo_ на момент написания была 3.0.3

Модуль состоит из трех базовых пакетов
