.. py:module:: xml.sax.saxutils

xml.sax.saxutils
================


.. py:method:: escape(<строка>[, <словарь>])

    заменяет символы `<,>,&` соответсвующими HTML эквивалентами. словарь позволяет дополнительный набор символов.

    >>> escape("""&<>" """)
    '&amp;&lt;&gt;" '


.. py:method:: quoteattr(<строка>[, <словарь>])

    функция аналогична :py:meth:`escape`, но доnолнительно заключает строку в кавычки или аnострофы. Если внутри строки встречаются только двойные кавычки, то строка заключается в аnострофы. Если внутри строки встречаются и кавычки, и аnострофы, то двойные кавычки заменяются НТМL-эквивалентом, а строка заключается в двойные кавычки. Если кавычки и аnострофы не входят в строку, то стро­ка заключается в двойные кавычки.


.. py:method:: unescape(<строка>[, <словарь>])

    заменяет HTML эквиваленты `&amp;, &lt;, &gt;` обычными символами. словарь задает дополнительный набор символов.
