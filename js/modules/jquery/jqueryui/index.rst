jQueryUI
========

jQuery UI – графический фреймворк для jQuery, представляющий собой библиотеку пользовательского интерфейса
с набором готовых решений для создания красивого и удобного интерфейса сайта.
Библиотека включает также набор готовых тем оформления, с возможностью тонкой настройки под любые требования.
