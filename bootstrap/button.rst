Кнопки
======

Кнопки можно создавать из тегов a, button, input

active
------

Активная кнопка

disabled
--------

Заблокированная кнопка

btn
---


btn-SW
------

* SW -
    * lg
    * sm
    * xs


btn-TYPE
--------

* TYPE -
    * default -
    * primary
    * success
    * info
    * warning
    * danger
    * link

btn-group
---------

Группа кнопок


btn-group-SW
------------

* SW -
    * lg
    * sm
    * xs

Размеры групп кнопок


btn-group-justified
-------------------

Растянуть кнопки


btn-group-vertical
------------------

Вертикальный блок кнопок


btn-toolbar
-----------

Панели кнопок
