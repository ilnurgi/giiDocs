Модальное окно
==============

Кнопке или ссылке добавляем:

* data-toggle="modal"
* data-target="#MODAL_ID"

Модальный блок должен иметь:

* классы modal fade
* id, для связи

Для закрытия окна, для элемента надо добавить:

* data-dismiss="modal"
