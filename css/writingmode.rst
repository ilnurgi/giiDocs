writtng-mode
============

Устанавливает положение текста

horizontal-tb
-------------

По умолчанию

.. code-block:: html

    <style>
        div {
            writing-mode: horizontal-tb;
        }
    </style>
    <div>ilnurgi.ru</div>

.. raw:: html

    <div style="writing-mode:horizontal-tb;border:1px solid black;padding:5px;">ilnurgi.ru</div>
