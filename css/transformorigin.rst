transform-origin
================

Точка, в которой применяется трансформация.

.. code-block:: text

    div {
        transform-origin: <horizontal-position> <vertical-position>;
    }

.. code-block:: css

    div {
        transform-origin: left top;
        transform-origin: 0% 100%;
        transform-origin: 10px -100px;
    }
