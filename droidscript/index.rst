Конспекты по DroidScript
========================

Официальная страница `DroidScript <http://droidscript.org/>`_

`Wiki <http://wiki.droidscript.me.uk/doku.php>`_, часто не доступна, но через прокси работает

Инстурмент позволяет разрабатывать Android приложения на языке JavaScript.

Приложение имеет следующие объекты:

    * `Images` - изображения, путь необходимо начинать с 'Img/'

        * Для задания иконки приложения, необходимо в асеты добаить картинку png с имененм вашего приложения
    
    * `Sounds` - аудио, путь необходимо начинать с 'Snd/'

    * `Html` - html, путь необходимо начинать с 'Html/'

    * `Misc` - разное, путь необходимо начинать с 'Misc/'

XMLHttpRequest также доступен для работы с сетью
