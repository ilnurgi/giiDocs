chmod
=====

Сменить права файла на указанную

chmod octal file

* octal - права (пользователь, группа, остальные)
  * - 4 – чтение (r)
  * - 2 – запись (w)
  * 1 – исполнение (x)
* file - путь к файлу

.. code-block:: sh

    # чтение, запись, исполнение для всех
    $ chmod 777
    
    # rwx для владельца, rx для группы и остальных
    $ chmod 755
